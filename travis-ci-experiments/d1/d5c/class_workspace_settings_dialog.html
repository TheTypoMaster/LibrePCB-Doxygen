Detailed Description

The WorkspaceSettingsDialog class.

This dialog class implements a GUI for all workspace settings. An instance of WorkspaceSettingsDialog is created in the class WorkspaceSettings. There must not exist more than one instance of this class at the same time in the same application instance!
