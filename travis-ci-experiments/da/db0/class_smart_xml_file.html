Detailed Description

The SmartXmlFile class represents a XML file and provides methods to load/save XML DOM trees (XmlDomDocument)

With parseFileAndBuildDomTree() the XML file can be parsed and a DOM tree is created. With save() the DOM tree can be saved back to the XML file.

Note
See class SmartFile for more information.
