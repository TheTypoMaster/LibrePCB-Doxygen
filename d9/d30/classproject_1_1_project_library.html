Detailed Description

The ProjectLibrary class.

Todo:

When adding new elements, the directory should be copied only to a temporary location. Then after saving the project, the directory must be copied to the project library's directory. The same applies to removed elements.

When removing elements, the objects should not be deleted! Undo commands need to add removed elements again, and the pointer to the object should remain the same.
